# 👨‍💼 Employee CRUD Operation

A basic **Employee Management System** built using PHP and MySQL to perform **CRUD operations** (Create, Read, Update, Delete).

---

## 📌 Features

* ➕ Add employee
* 📋 View employee list
* ✏️ Edit employee details
* ❌ Delete employee

These are the core operations used in most database applications. ([SymaDev][1])

---

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML, CSS
* XAMPP

---

## ⚙️ Setup

1. Clone the repository

```bash
git clone https://github.com/sanjaidevan/Employee-CRUD-operation.git
```

2. Move to XAMPP folder

```bash
C:/xampp/htdocs/
```

3. Start Apache & MySQL

4. Create database

```sql
CREATE DATABASE db_employee;
```

---

## ▶️ Run Project

Open in browser:

```bash
http://localhost/Employee-CRUD-operation/
```
